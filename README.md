# Cabinet Robert DOSSOU - International Law Firm

**Repository Name:** `cabinet-robert-dossou-showcase`

## Description
This project is a high-level, international, and bilingual corporate showcase website for **Cabinet Robert DOSSOU**, a prestigious law firm specializing in international law, arbitration, and constitutional expertise. 

The application is designed to inspire credibility, diplomacy, and global authority. It features a minimalist and structured UI, a strict grid system, and precise typography (Fraunces & Public Sans).

## Key Features
- **Multi-Page Architecture**: Dedicated pages for Heritage, Services, Expertise, Publications, and Contact.
- **Bilingual Support**: Full French and English localization with a seamless language switcher.
- **Swiss Design Aesthetic**: Deep Forest Green (#0E2F22), Matte Gold (#D4AF37), and Pearl Gray (#F2F2F2) color palette.
- **Interactive Elements**:
  - Animated wireframe globe representing global reach.
  - Horizontal scrollable timeline for the firm's heritage.
  - Stylized international expertise map with animated connections.
  - Refined contact form with floating labels and institutional styling.
- **Detailed Presentation**: Comprehensive sections detailing the firm's history and the distinguished career of Maître Robert DOSSOU.
- **Publications Section**: A dedicated space for institutional reports and legal insights.

## Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **Routing**: React Router Dom

## Structure
- `/src/App.tsx`: Main application logic and routing.
- `/src/translations.ts`: Centralized bilingual content management.
- `/src/index.css`: Global styles and Swiss Design theme variables.
- `/src/main.tsx`: Application entry point.

## Installation & Development
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Build for production: `npm run build`
